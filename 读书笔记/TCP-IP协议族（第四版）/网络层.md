# 网络层简介
网络层解决经由多条链路的交付问题，负责主机到主机的交付，并且在分组经过路由器时负责为其选择路由。

### 分组交换
在分组交换网中，源点在传送之前先要将报文分割成便于管理的分组，这些分组在到达终点之后被重新组装。
目前分组交换网为分组选择路由的方式有数据报方式和虚电路方式。
