# 数据库

## SQLITE

```
SQLite 是一个软件库，实现了自给自足的、无服务器的、零配置的、事务性的 SQL 数据库引擎。
SQLite 是在世界上最广泛部署的 SQL 数据库引擎。
SQLite 源代码不受版权限制。
```
